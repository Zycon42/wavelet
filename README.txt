PROJEKT DO PREDMETU ZPO/2013L
Komprimace obrazu pomoci waveletove transformace

Zavislosti:
gcc >= 4.6 || msvc >= 11
opencv
googletest (nepovinne)

Preklad:

linux
$ mkdir build && cd build
$ cmake ..
$ make

msvc
$ mkdir build && cd build
$ cmake -G "Visual Studio 11" ..
$ prelozit v msvc

Spusteni:
binarky jsou v build/bin
